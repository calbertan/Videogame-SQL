This Page inserts entries to either Platform, Ip, or Consumer

Inserting into Platform


Columns in Platform table are :
Name of platform
Year it was discontinued (leave empty if not yet discontinued)
Name of company that manufactured it

Note: Both the name of the platform and the name of the company cannot be empty






Inserting into Ip


Columns in Ip table are :
Intellectual Property Name

Note: IP Name cannot be empty




Inserting into Consumer


Columns in Consumer table are :
Name of person
Their consumer ID
Their age

Note: Consumer ID cannot be empty
